sendMail()

sendMail(String  $recipient, String  $subject, String  $message, String  $fromName, String  $fromAddress, String  $attachments = null) : String

A function that will send a confirmation email to the user.

The sendMail function takes on a number of params and sends and email to a receipient.

Parameters

String $recipient

This can be a String or Array, the Array should be ; delimited
String $subject

The subject for the email
String $message

The message to send to the Receipient
String $fromName

The name of the person sending the message
String $fromAddress

The address of the person sending the message
String $attachments

An Array of file paths to be attached in the form array ( array( "filename", "path" ) )

Returns

String —

OK or Failed
